# Docusaurus Configuration File

Docusaurus configuration file is a file named `docusaurus.config.js` at the root of your project. It is a Node.js module that exports an object.

```javascript
module.exports = {
  // ...
};
```

## Properties

### `title`

- Type: `string`
- Default: `undefined`

The title of your site. It will be used as the title of the website and the default title for all pages.
